## 项目目标
- 做一个工具，用户可以用这个工具来写简历

## 功能列表
1. 添加简历内容，包括个人信息、工作经历、学习经历、获奖情况、项目经历和联系方式。
2. 更新简历内容
3. 选择简历模板
4. 预览简历
5. 发布简历

## 原型图|分模块|分区
1. 顶栏。包含 logo 和一些按钮。
2. 编辑区。包含一组按钮和一些表单。
3. 预览区。展示 HTML，可切换模板。
